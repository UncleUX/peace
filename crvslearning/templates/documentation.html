Documentation CRVS Learning

Bienvenue dans la documentation complète de la plateforme CRVS Learning, votre solution tout-en-un pour l'apprentissage en ligne.

Cette documentation est en constante évolution. N'hésitez pas à nous faire part de vos suggestions.

À propos de CRVS Learning

CRVS Learning est une plateforme d'apprentissage en ligne complète qui permet :

  • La création et la gestion de cours en ligne
  • L'évaluation des apprenants
  • Le suivi des progrès
  • La gestion des utilisateurs et des rôles

Démarrage rapide

Commencez avec CRVS Learning en suivant ces étapes simples :

1

Installation

# Cloner le dépôt
git clone [url-du-depot]

# Installer les dépendances
pip install -r requirements.txt

# Configurer les variables d'environnement
cp .env.example .env

# Effectuer les migrations
python manage.py migrate

# Créer un superutilisateur
python manage.py createsuperuser

# Lancer le serveur
python manage.py runserver
2

Première connexion

  1. Ouvrez votre navigateur à l'adresse http://localhost:8000
  2. Connectez-vous avec vos identifiants d'administration
  3. Accédez au tableau de bord d'administration

Gestion des utilisateurs

CRVS Learning propose différents rôles utilisateurs :

Apprenant

Peut suivre des cours, passer des évaluations et voir sa progression.

Formateur

Peut créer et gérer des cours, évaluer les apprenants.

Administrateur

Accès complet à toutes les fonctionnalités de la plateforme.

Gestion des cours

Créez et gérez facilement vos cours en ligne :

Seuls les utilisateurs avec les droits appropriés peuvent créer et gérer des cours.

Structure d'un cours

Un cours est composé de :

  • Catégories
  • Leçons
  • Exercices
  • Évaluations

Déploiement

Guide de déploiement de CRVS Learning en production :

Configuration requise

  • Python 3.8+
  • PostgreSQL 12+
  • Redis (pour le cache et les files d'attente)
  • Nginx ou Apache

Variables d'environnement

Créez un fichier .env à la racine du projet :

DEBUG=False
SECRET_KEY=votre_cle_secrete
DATABASE_URL=postgres://user:password@localhost:5432/crvslearning
REDIS_URL=redis://localhost:6379/0
ALLOWED_HOSTS=.votredomaine.com,.votredomaine.fr
